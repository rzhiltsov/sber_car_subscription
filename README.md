# Описание
Проект содержит модель машинного обучения, которая предсказывает, что клиент совершит целевое действие на сайте СберАвтоподписки
на основе данных о посещении сайта.
Здесь происходит исследование данных, обучение модели и конфигурация микросервиса для работы.

# Стек:
+ Python 3.13
+ Pandas
+ Sklearn
+ Joblib
+ Jupyter Notebook
+ FastAPI
